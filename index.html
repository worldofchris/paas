REST API

Of course, if it's SaaS it's all about teh API.

Simply ask for a pun on any given subject, for example eggs:

dadjokes.org/pun/eggs

And you'll get back a Dad Jokes quality pun:

That's eggcellent!

I know it is!

Want some JSON with that sir?

But we don't just has text, need puns in your apps? REST easy, you'll be the JSON King with our JavaScipts flavour response

dadjokes.org/pun/eggs/json

Yields: 

{
    "subject": "eggs",
    "pun": "That's eggcellent!"
}

What if we don't has it?

Say you want a pun for say, goats, but we haven't thought of one one yet? Don't worry, take a ticket and we'll let you know when your pun is ready...
